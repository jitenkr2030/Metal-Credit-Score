API Documentation

Comprehensive reference for all MCS API endpoints.

Authentication

All API requests require authentication using Bearer token in the Authorization header.

curl -X GET "https://api.mcs.com/v1/mcs/score/user_123456" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"

API Endpoints

GET /api/v1/mcs/score/:userId

Retrieve the Metal Credit Score for a specific user based on their asset portfolio.

Parameters:
  • userId (path) - User identifier
  • includeComponents (query, optional) - Include score breakdown
  • history (query, optional) - Include historical data
Response:

GET /api/v1/mcs/score/:userId/history

Get historical score data for trend analysis and monitoring.

Query Parameters:
  • period - Time period (7d, 30d, 90d, 1y)
  • interval - Data interval (daily, weekly, monthly)
POST /api/v1/webhooks

Register webhook endpoints to receive real-time score updates and notifications.

Webhook Events:
  • score.updated - User's score has changed
  • portfolio.changed - Portfolio assets have been updated
  • score.analyzed - New score analysis completed
  • alert.triggered - Risk alert or threshold exceeded

Integration Examples

Complete code examples for common integration scenarios.

Node.js Integration
const MCS = require('@mcs/sdk-js');

const client = new MCS({
  apiKey: 'your-api-key',
  environment: 'production'
});

// Get user score
const score = await client.getScore('user_123456');
console.log(`Score: ${score.finalScore}`);

// Update portfolio
await client.updatePortfolio({
  userId: 'user_123456',
  assets: [
    { type: 'gold', amount: 125.5 },
    { type: 'silver', amount: 450 }
  ]
});
Python Integration
from mcs import MCSClient

client = MCSClient(
    api_key='your-api-key',
    environment='production'
)

# Get user score
score = client.get_score('user_123456')
print(f"Score: {score.final_score}")

# Update portfolio
portfolio_data = {
    'user_id': 'user_123456',
    'assets': [
        {'type': 'gold', 'amount': 125.5},
        {'type': 'silver', 'amount': 450}
    ]
}
client.update_portfolio(portfolio_data)

Webhook Handler
app.post('/webhook/mcs', (req, res) => {
  const { event, data } = req.body;
  
  switch (event) {
    case 'score.updated':
      console.log(`Score updated: ${data.finalScore}`);
      // Update user interface
      updateUserInterface(data.userId, data.finalScore);
      break;
      
    case 'alert.triggered':
      console.log(`Alert: ${data.message}`);
      // Send notification
      sendNotification(data.userId, data.message);
      break;
  }
  
  res.status(200).json({ received: true });
});
